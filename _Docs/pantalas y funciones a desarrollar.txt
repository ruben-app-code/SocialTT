# DOCUMENTO FUNCIONAL – NUESTROMURO.COM

## 1. VISIÓN GENERAL

NuestroMuro es una plataforma web orientada a creadores de contenido (principalmente de TikTok) que permite:

* Centralizar sus redes sociales
* Mostrar horarios de actividad globales
* Registrar crecimiento de seguidores
* Interactuar con seguidores (comentarios, encuestas)
* Publicar próximos eventos (lives)
* Recibir mensajes sin exponer datos personales
* Monetizar su perfil (suscripciones, anuncios, servicios)

El sistema también permite a los visitantes descubrir creadores por temas, país, nivel y estado de sus cuentas.

---

## 2. TIPOS DE USUARIO

### 2.1 Visitante (no autenticado)

* Puede ver perfiles públicos
* Puede explorar directorios (tema, país)
* Puede votar y comentar solo si se registra

### 2.2 Follower (usuario registrado)

* Puede seguir creadores
* Puede comentar
* Puede votar en encuestas
* Puede donar
* Puede enviar mensajes (si está habilitado)

### 2.3 Creador

* Administra su perfil
* Registra redes sociales
* Define horarios
* Publica encuestas
* Publica próximos lives
* Actualiza seguidores
* Accede a monetización

### 2.4 Administrador

* Gestiona usuarios
* Gestiona anuncios
* Gestiona configuraciones
* Supervisa actividad

# 🔁 6. FLUJOS IMPORTANTES

---

## Registro de creador

1. Se registra
2. Verifica email
3. Configura perfil
4. Agrega redes
5. Define horarios

---

## Actualización de seguidores

1. Entra al dashboard
2. Sistema detecta tiempo sin actualizar
3. Solicita actualización
4. Guarda snapshot

---

## Encuesta

1. Creador crea encuesta
2. Usuarios votan
3. Sistema guarda votos
4. Se muestran resultados

---

## Verificación de cuenta

1. Usuario registra cuenta
2. Marca método
3. Sistema guarda evento

---

# 🧠 7. REGLAS GENERALES

* No duplicar lógica
* Separar estado actual vs historial
* Usar timezone real
* No usar offsets manuales
* Mantener datos históricos
